# order of progression  

## `data_types.c`  
- identify basic data types in c.
example:  
```  
char  
bool // not really but you can try printing out true or false.
int  
float  
double 
char * //string  
```  
- get the size of each data type using `sizeof()`.
- create a variable for each data type.
- assign a appropriate value to each variable.
- print out the value of each variable using `printf()`.

## `output.c`
- get familiar with `printf()`
- learn about format specifiers and how to print out each data types.
- assign `3.1415926535` to a double and print only output the first 2 decimals, ex: `3.14`
- learn and try out escape sequence `\`.

## `string.c`  
- learn the difference between `char` and `char *`.
- `'` vs `"`.
- experiment with strings learn to concat or copy and etc.  

## `function.c`
- learn about return types.
- learn about arguments.
- learn about and experiment with type casting.

**exercise**  
create a function that adds two numbers and returns a float as its return value.

## header files
- read on how header file works.
- take a look in `stdio.h`

**exercise**  
create a dir named `header` and create these files; `add.c`, `add.h`, and `main.c`, figure out on how you can call the `add()` function from `add.c` in the `main()` function in `main.c`

## `operators.c`
- learn and try out C's built in operators.
- learn the difference between prefix `++x` vs postfix `x++`.

**exercise**  
try recreating at least 5 simple formulas using the operators you learned about.

## `control_flow.c` 
- learn about `if` statements and `switch` statements. 
- experiment with the `if` and `switch` statements.

## `loops.c`
- learn about `for`, `while`, and `do while` loops.
- learn the difference and use case for those loops.
- experiment with those loops.

**exercise**  
create `Fizz Buzz` 
- print `Fizz Buzz` if the number is divisible by `3` and `5`.
- print `Fizz` if the number is divisible by `3`.
- print `Buzz` if the number is divisible by `5`.
